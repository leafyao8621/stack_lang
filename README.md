# Stack Lang

A stack based interpreted language. Uses RPN

## v2
### Operators implemented:
* \+
* \-
* \*
* /
* %
* =
* \>
* \<
* \>=
* \<=
* ==
* !=
* \>\>
* \<\<
* ~
* !
* &
* &&
* |
* ||
* ^
* ++
* +=
* --
* -=
* [] (Not yet implemented)

### Commands Implemented:
* _input (Not yet implemented)
* _print
* _println
* _if
* _else
* _end
* _while
* _do
* _def (Not yet implemented)
* _begin (Not yet implemented)
